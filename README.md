
# 📊 My Excel Projects Repository
---

## 📌 Overview

This repository contains my **Week 1 Excel project** completed during the Data Technician Bootcamp.

I worked with multiple datasets — including retail sales, regional product performance, and bike sales demographics — to build practical skills in **data cleaning, analysis, and visual storytelling using Excel**.

---

## 🚀 Project Highlights

- 📊 Data analysis using real-world datasets  
- 🧹 Cleaning and preparing raw data for reporting  
- 📈 Building PivotTables for multi-level insights  
- 🧠 Applying Excel formulas for automation  
- 📉 Creating charts and dashboards for visualisation  

---

## 🧠 Skills Demonstrated

### 🔢 Excel Formulas
- `SUM`, `SUMIF` — totals and conditional calculations  
- `AVERAGE`, `AVERAGEIF` — grouped averages  
- `VLOOKUP` — data retrieval across tables  
- `DATE`, `MONTH`, `YEAR` — date extraction  
- `UNIQUE` — identifying distinct values  
- `SWITCH` — categorising sales levels  

---

### 🧹 Data Cleaning
- Sorting and filtering datasets  
- Standardising formats  
- Organising age groups and transaction IDs  
- Preparing clean data for PivotTables  

---

### 📉 Data Visualisation
Created visualisations including:

- 📊 Bar & column charts for comparisons  
- 📈 Line charts for trends  
- 🥧 Pie & donut charts for demographics  
- 📦 Stacked charts for category breakdowns  

---

## 📁 Datasets Used

- 🛍️ Retail Sales Dataset — formulas & filtering practice  
- 🌍 County Sales Dataset — PivotTables & classification  
- 🚴 Bike Sales Pivot Dataset — demographic insights  
- 📊 Bike Sales Visualisation Dataset — dashboards & charts  

---

## 📝 Key Takeaways

- Improved ability to clean and structure raw data  
- Strengthened PivotTable and analytical thinking skills  
- Automated calculations using Excel formulas  
- Translated data into meaningful business insights  
- Developed dashboard-style reporting techniques
